What is Simply Explain?

Simply Explain is a chrome extension that leverages Open AI's GPT models to assist users with reading and understanding text via Text-To-Speech (TTS).


With the expanding capabilities of Large-Language-Models (LLM) it is possible to provide more accessibility through assistive technology such as 'Simply Explain'. As LLM's are known to have "hallucinations", there is always a chance the response may be incorrect or nonsensical.

Controls

1 - Select Text

2 - Right-Click (Open Context Menu)

3 - Left-Click Explain


To use Simply Explain please set an API Key which can be found at https://platform.openai.com/account/api-keys. The API Key will be saved locally on your PC to be used to send requests to the GPT model.
